# Project Rules

## Forms

1. All forms must use React Hook Form for state management and Zod for schema validation.

2. Every form must display inline validation messages for every invalid field before submission.

3. Password confirmation fields must always validate against the original password.

## Accessibility

4. Every input must have an associated label and support keyboard navigation.

5. Forms must use semantic HTML elements and accessible validation messages.

## AI Workflow

6. Before implementing a feature, generate a plan and verify requirements with AI before writing code.

7. Every AI-generated feature must include a verification step or tests before it is considered complete.

8. Never merge AI-generated code without manually reviewing validation logic and edge cases.
