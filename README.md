## Project Description

**FlyRank** is a capstone project that ingests data, applies ranking logic, and presents sorted results through a Node.js application.

The project demonstrates end-to-end software development: data handling, business logic, API or CLI design, version control, and deployment-ready project structure. It is built as a learning portfolio piece showcasing clean code, maintainability, and professional Git practices.

---

## Goals

- **Core functionality** — Implement a ranking pipeline that scores, sorts, and filters items based on defined criteria.
- **Clean architecture** — Organize code into clear modules with single responsibilities and readable async flows.
- **Professional workflow** — Use Git, GitHub, and [Conventional Commits](https://www.conventionalcommits.org/) for traceable development.
- **Maintainability** — Follow ES6+ standards, async/await patterns, and clean code principles.
- **Documentation** — Keep setup, usage, and repository layout documented for reviewers and collaborators.
